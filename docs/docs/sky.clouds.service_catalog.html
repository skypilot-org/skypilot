sky.clouds.service_catalog.common module¶

Common utilities for service catalog.

class sky.clouds.service_catalog.common.InstanceTypeInfo(cloud: str, instance_type: str, accelerator_name: str, accelerator_count: int, memory: float)[source]¶

Bases: tuple

Instance type information.

  • cloud: Cloud name.

  • instance_type: String that can be used in YAML to specify this instance type. E.g. p3.2xlarge.

  • accelerator_name: Canonical name of the accelerator. E.g. V100.

  • accelerator_count: Number of accelerators offered by this instance type.

  • memory: Instance memory in GiB.

  • price: Regular instance price per hour.
