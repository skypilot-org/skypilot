class sky.data.storage.Storage(name, source, stores=None, persistent=True)[source]¶

Bases: object

Storage objects handle persistent and large volume storage in the sky.

Users create Storage objects by defining the storage object name and the source, where the data originally comes from. Power users can specify their pre-initialized stores if their data is already on the cloud.

Typical Usage: (See prototype/examples/playground/storage_playground.py)

storage = Storage(name=’imagenet-bucket’, source=’~/Documents/imagenet’)

# Move data to S3 storage.get_or_copy_to_s3()

# Move data to Google Cloud Storage storage.get_or_copy_to_gcs()

# Delete Storage for both S3 and GCS storage.delete()

  • source (str) –

  • stores (Optional[Dict[StorageType, AbstractStore]]) –
