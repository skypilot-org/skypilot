üöÄ Launch the System

  1. Start the Evaluation Head: 
    # With API server for auto-discovery
sky launch -c eval-head configs/eval_head.yaml \
  --env GAME_SERVER_PREFIX=exp-game
  2. Get the Eval Head Endpoint: 
    EVAL_HEAD_ENDPOINT=$(sky status --endpoint eval-head 8080)
  3. Launch Game Servers: 
    # Launch multiple game servers with matching prefix
sky launch -c exp-game-1 configs/game_server.yaml \
  --env EVAL_HEAD_ENDPOINT=$EVAL_HEAD_ENDPOINT

sky launch -c exp-game-2 configs/game_server.yaml \
  --env EVAL_HEAD_ENDPOINT=$EVAL_HEAD_ENDPOINT
  4. Monitor Status: 
    sky status  # Check all clusters

üîÑ Restart & Update

To update code without terminating clusters:

# Restart eval head
sky cancel -ay eval-head
sky exec eval-head configs/eval_head.yaml \
  --env GAME_SERVER_PREFIX=exp-game

# Restart game server
sky cancel -ay exp-game-1
sky exec exp-game-1 configs/game_server.yaml \
  --env EVAL_HEAD_ENDPOINT=$EVAL_HEAD_ENDPOINT
üí° Tips:
  • Game servers with prefix exp-game-* are auto-discovered
  • The dashboard auto-refreshes every 30 seconds
  • Discovery checks for new clusters every 30 seconds
  • Game servers run on spot instances for cost savings
